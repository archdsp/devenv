# How to setup
## Initial Setup
```sh
git fetch --all
git reset --hard origin/<branch-name>
```

# Setting list
## Shell
* bash runtime configuration : .bashrc
* user environment : bash_profile
* user alias : bash_alias

## IDE
* vim : .vimrc
* emacs : .emacs

## Git
* .gitconfig
* .git-template
* .gitmessage : Commit message template
* .gitignore
* .gitattribute
